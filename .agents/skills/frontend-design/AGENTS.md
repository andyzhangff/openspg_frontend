# Frontend Design Skill

## 概述

此 skill 用于指导创建独特、生产级的前端界面，避免通用"AI 生成"的平庸美学。

## 使用场景

- 构建网页组件、页面、海报或应用程序
- 网站、落地页、仪表盘设计
- React/Vue 组件开发
- HTML/CSS 布局
- UI 样式美化

## 核心原则

### 1. 设计思维 (Design Thinking)
- **目的**: 界面解决什么问题？谁在使用？
- **风格**: 选择一个极端方向（极简主义、极繁主义、复古未来主义等）
- **差异化**: 什么让设计令人难忘？

### 2. 美学准则
- **字体**: 选择独特有趣的字体，避免通用字体（Arial, Inter 等）
- **配色**: 使用 CSS 变量保持一致性，主色+强调色优于均分调色板
- **动效**: 使用 CSS 动画，高影响力时刻使用编排好的入场动画
- **空间构图**: 非对称、重叠、对角线流、打破网格
- **背景细节**: 渐变网格、噪点纹理、几何图案、颗粒叠加

### 3. 禁止事项
- ❌ 通用 AI 美学（紫色渐变白色背景）
- ❌ 过度使用的字体（Space Grotesk 等）
- ❌ 可预测的组件模式
- ❌ 缺乏上下文特色的设计

## 应用示例

当前项目 OpenSPG Schema 编辑器采用了：
- **风格**: 复古未来主义赛博朋克
- **字体**: Orbitron (显示) + Rajdhani (正文)
- **配色**: 霓虹青 (#00f0ff) + 紫 (#b829dd)
- **特效**: 3D 透视网格、扫描线、噪点纹理、玻璃拟态
